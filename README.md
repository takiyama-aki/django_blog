## メモ
- プロジェクト作成

`django-admin startproject [プロジェクト名]`

- アプリケーション作成

`python manage.py startapp [アプリケーション名]`

- サーバー起動

`python manage.py runserver`

- データベース作成

`python manage.py makemigrations`
