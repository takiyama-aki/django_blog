## メモ
- プロジェクト作成

`django-admin startproject [プロジェクト名]`
  <br>
  
- アプリケーション作成

`python manage.py startapp [アプリケーション名]`
  <br>
  
- サーバー起動

`python manage.py runserver`
  <br>
  
- データベース作成

`python manage.py makemigrations`
